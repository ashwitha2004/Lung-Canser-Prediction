The project demonstrates how machine learning techniques can be used to support early-stage lung cancer risk assessment.

## Project Overview

Lung cancer remains one of the leading causes of cancer-related deaths globally. Early detection can significantly improve survival rates.  
This project builds a predictive system that analyzes medical and lifestyle factors to assess whether a patient is at risk of lung cancer.  

The repository includes:

- Dataset and preprocessing workflow  
- ML model training (classification)  
- Django application for interactive prediction  
- Preconfigured environment using `requirements.txt`  

---

## Features

- Intuitive web interface for entering patient details  
- Machine learning pipeline for training classifiers  
- Clean dataset folder structure  
- Django-based backend for real-time predictions  
- Simple deployment using `runserver`
